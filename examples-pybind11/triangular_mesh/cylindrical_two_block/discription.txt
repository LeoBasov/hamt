
Simulation Domain
-----------------
The simulation domain consists of two boxes stacked on one another.

            T = T1
          ++++++++++
          +        +
dT/dx = 0 + labda2 + dT/dx = 0
          +        +
          ++++++++++
          +        +
dT/dx = 0 + labda1 + dT/dx = 0
          +        +
          ++++++++++
            T = T1

Simulation parameters
---------------------
T1 = 300 K
T2 = 100 K
labda1 = 20 W m^-1 K^-1
labda1 = 50 W m^-1 K^-1
